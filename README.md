## How to Combine ROMs

Once all three ROMs have been added and edited individually, follow these steps to **combine them** into the **Sonic Classic Trilogy: Ultimate Complete** ROM using **Esrael Sonic Editor II**.

### 🛠️ Combination Steps

1. Open **Esrael Sonic Editor II**.
2. Navigate to the **Trilogy / Multi-ROM** menu or section.
3. Click **New Project → Sonic Classic Trilogy: Ultimate Complete**.
4. Add each ROM in the following order:

   | Order | ROM | File |
   |-------|-----|------|
   | 1st | Sonic The Hedgehog | `Sonic The Hedgehog (USA, Europe).bin` |
   | 2nd | Sonic The Hedgehog 2 | `Sonic The Hedgehog 2 (World).bin` |
   | 3rd | Sonic 3 & Knuckles | `Sonic The Hedgehog 3 & Knuckles (World).bin` |

5. Configure the **menu / selector screen** settings (title, background, music, etc.).
6. Set the output file name:
   ```
   Sonic Classic Trilogy - Ultimate Complete (Sega Genesis).bin
   ```
7. Click **Build / Combine** to generate the final combined ROM.
8. Wait for the process to complete — the editor will notify you when the ROM is ready.
9. Test the combined ROM in your preferred **Sega Mega Drive / Genesis emulator**.

> ⚠️ **Important:** Always use the ROMs in the order listed above (Sonic 1 → Sonic 2 → Sonic 3 & Knuckles) to ensure the combined ROM boots correctly.

> ✅ **Tip:** The combined ROM will include a game selection menu that allows players to choose which Sonic game to play directly from a single ROM file.
